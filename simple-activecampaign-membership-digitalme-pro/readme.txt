Allow or disallow a specific ActiveCampaign tag to either show or not show a page

== Description ==

This lightweight plugin lets you control the visibility of specific pages based on a visitor's ActiveCampaign tag.

Whether you're building a membership site, offering gated content, or customizing user experiences, this plugin gives you precise control over who sees what â€” using the power of ActiveCampaign tag.

Simply choose one tag, and decide whether users with (or without) those tag can view a particular page.
